# EasyBank

A static web prototype for an online banking dashboard built with vanilla HTML, CSS, and JavaScript.

## Project Overview

This project demonstrates a polished banking experience with a login flow, dashboard, transfer workflow, bill pay, account overview, and support chat.

### Included files

- `easybank_index.html` — Main prototype page containing the app structure and screens.
- `easybank_style.css` — Styling for the interface, including cards, forms, buttons, tables, and responsive components.
- `easybank_app.js` — App logic for login, navigation, transfer flow, bill pay, account rendering, session timeout modal, and support interactions.

## Features

- Login prototype with error handling and session state 
- Dashboard with quick actions and recent transactions
- Multi-step transfer flow: select type, enter details, confirm, and success screen
- Bill pay scheduler with sample payees and success confirmation
- Account overview with account cards and transaction search
- Support chat UI with instant replies and email form
- Session timeout modal to simulate inactivity logout behavior

## Usage

1. Open `easybank_index.html` in a web browser.
2. Enter any username and password to sign in.
3. Use the dashboard to navigate transfers, bill pay, accounts, and support.

> Tip: Use `wrong` as the password to view the login error state.

## Notes

- This is a front-end prototype only and does not connect to a real backend.
- The app runs entirely in the browser and uses static sample data.
